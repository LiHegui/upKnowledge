## 学习节奏建议

| 原则 | 说明 |
|------|------|
| **每日 1.5-2h** | 不贪多，保持可持续节奏 |
| **先读再练** | 先读知识库 Q&A → 合上答案自己复述 → 写代码验证 |
| **每周日复习** | 快速过本周所有题，标记仍然不熟的 |
| **面试官模式** | 每阶段结束后用面试官模式自测，防止"看了等于会了" |
| **笔记留痕** | 每天填写学习记录，下次对话可以基于笔记继续推进 |
| **Node 快通** | 你有 Node 基础，Day 41 重点只看面试高频考点 |
