# Contentful
>Contentful 是一个无头内容管理系统 (CMS)。您可以将内容（无论是文本、图像还是视频）上传到 Contentful，然后可以根据需要对其进行组织和编辑。
- 无头内容管理系统（ headless content management system (CMS)）

# Contentful APP

npx create-contentful-app my-first-app
